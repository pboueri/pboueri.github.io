Core foundations of software engineering -- abstraction, collaboration, correctness, and composition -- still matter, despite significant changes in how we build software today.

This presentation examines what mature software projects should resemble in the future, including their definition, construction, and evolution. The session features prepared demonstrations and materials, with dedicated time for audience discussion and questions.

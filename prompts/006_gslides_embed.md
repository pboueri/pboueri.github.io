# Google Slides Embed for Talks

## Background
I want to be able to iframe in a particular Google presentation from a URL. It should be in presentation mode if possible and should be right alongside talks.

## Requirements
- Embed Google Slides via iframe in presentation/preview mode
- Reusable layout template that reads slides URL from frontmatter
- Should sit alongside other talks in the talks section
- Responsive 16:9 aspect ratio embed
- Link to the event page (Meetup, etc.)
- Clean styling consistent with the rest of the site
- Homepage Talks section should dynamically generate links from talk content, sorted most recent first, max 3 shown with a "View all talks" link
